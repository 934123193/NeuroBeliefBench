# NeuroBeliefBench

This repository accompanies a manuscript currently under review.

NeuroBeliefBench is a matched-evidence benchmark for evaluating how patient- and clinician-derived belief cues influence medical large language model diagnostic behavior under unchanged factual clinical evidence.

The full codebase, prompt templates, evaluation scripts, and non-sensitive derived materials will be released upon manuscript acceptance. Custom code central to the main analyses can be made available to editors and reviewers upon reasonable request during peer review.

Raw clinical records and original narrative texts are not publicly available because they contain sensitive patient information and are subject to institutional data-governance restrictions.

Planned release materials include:

- prompt templates;
- belief-cue definitions;
- diagnostic matching scripts;
- confidence–accuracy analysis scripts;
- statistical analysis scripts;
- aggregate result tables;
- figure source data;

A formal citation and DOI will be added after publication.
